# Papyrus - Seu Gerenciador de Estudos

Este projeto está sendo desenvolvido como parte da avaliação da disciplina de **Experiência Criativa: Projetando Soluções Computacionais**. Trata-se de uma aplicação web para o gerenciamento da vida turbulenta de todos os estudantes. 

---

##  Sobre o Projeto

O **Papyrus** permite que o usuário gerencie os temas que estão sendo estudados, as disciplinas da faculdade, os horários do dia a dia e mais tópicos de extrema importância na vida acadêmica. A proposta foca em centralizar a organização em uma interface intuitiva e funcional.
