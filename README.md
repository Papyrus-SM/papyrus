# Papyrus - Organizador de Estudos

O **Papyrus** é uma aplicação web feita para ajudar estudantes a organizarem suas rotinas, matérias e prazos de forma inteligente. Este projeto separa totalmente o **Backend** (nossa inteligência e banco de dados) do **Frontend** (nossa interface visual).

## Tecnologias Principais
* **Frontend:** React + Vite + Tailwind CSS.
* **Backend:** PHP Puro (API JSON) + MySQL.
