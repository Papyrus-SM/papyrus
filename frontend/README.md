# Papyrus - Frontend

Esta é a interface do usuário do Papyrus. Aqui focamos na experiência do estudante, criando uma aplicação rápida e moderna (SPA) usando React.

## O que estamos usando?
* **React + Vite:** Para uma inicialização e recarregamento super rápidos.
* **Tailwind CSS:** Para estilização moderna direto no HTML/JSX.

## Comandos Iniciais
Se você acabou de clonar o projeto, rode os seguintes comandos dentro desta pasta:

```bash
# Instalar as dependências
npm install

# Rodar o projeto em modo de desenvolvimento
npm run dev
